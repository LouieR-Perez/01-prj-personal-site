Intel Website Localization

In this project, I enhanced my Intel website by implementing Bootstrap to streamline layout design and responsiveness, making it easier to align and customize sections using the 12-column grid system. I added a subscription form to improve user engagement and integrated localization with RTL (right-to-left) functionality to support languages that require it. I also focused on accessibility improvements, using tools like Lighthouse and AI assistance to identify and fix issues, ensuring the site is inclusive and user-friendly across devices, cultures, and abilities.
